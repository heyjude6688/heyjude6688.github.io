hexo + next + github + 私有域名

发表于 2018-06-02 | 更新于 2018-06-03 | 分类于 technology |
第一次搭建博客写一下简短搭建过程！

准备工作

git：git官网
nodejs：nodejs官网
GitHub：GitHub官网
DNSPod：DNSPod官网
域名：国内的需要备案，也可以买国外的，我选择 Namecheap

安装过程

  • 安装git和nodejs

  • 调出控制台输入以下命令

    安装成功

  • 安装hexo：运行 Git Bash 输入 npm install hexo-cli -g
  • 在合适的位置新建一个文件夹名为 Blog 用于存放博客，打开Blog右键运行 Git Bash
    输入 npm install 然后输入 hexo s ，在浏览器中输入 http://localhost:4000/ 就可以看到效果了

配置过程

  • 登陆DNSPod，添加域名绑定，成功后进入域名管理，复制两条记录类型为 NS的记录值。
  • 登陆namecheap，查看你买的域名，点击右侧 MANAGE ，NAMESERVERS 选择 Custom DNS,
    添加两条上面复制的记录值，保存。
  • 登陆GitHub，新建仓库名为 username.github.io ，其中 username 一定要为GitHub用户名。在新建仓库根目录新建一个文件 CNAME ，里面填上你购买的域名，保存。
  • 再次登陆DNSPod，在域名解析界面选择你的域名添加两条记录
    主机记录 @ 记录类型 CANME 记录值 username.github.io
    主机记录 www 记录类型 CANME 记录值 username.github.io
    保存。
